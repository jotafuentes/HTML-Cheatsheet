Basic HTML Semantic Tags For Content Structure

TAG NAME DEFINITION
<header> header Top part of a site, used for basic intro information, such as main Heading, Logo etc
<nav> Nav Used to navigation part of a site. Mainly includes hyperlinks, as internal so external
<main> Main Main part of a site, which consict main content
<footer> Footer Most bottom part of a side, mostly used for "tech" information such as contacts, additional links etc
<article> Article Consist main usefull information for viewers. May include any type of content: from text to media
<aside> Aside Commonly consist additional to main Article info
h1,h2,h3... H1 Headings for all type of content. Usefull for SEO
